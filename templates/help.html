Help & User Guide

About the System

The Student Performance Prediction System helps students estimate their likely final course grades using continuous assessment records such as assignments, quizzes and projects.

How to Use the System

  1. Enter the Course Code and Course Name.
  2. Enter assignment, quiz and project scores.
  3. Multiple scores should be separated with commas.
  4. Fractions such as 78/90 are accepted and automatically converted into percentages.
  5. If an assessment was missed, enter 0.
  6. If a category is unavailable, leave it blank.
  7. Click Predict Grades.
  8. View your predicted score, grade, remarks and recommendations.
